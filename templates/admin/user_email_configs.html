Como funciona o Item 34 - E-mails por Usuário:
  • Envio pelo usuário que preencheu: Quando um relatório for criado, será enviado usando a configuração SMTP do usuário que o preencheu
  • Envio pelo aprovador: Quando um relatório for aprovado, será enviado usando a configuração SMTP do aprovador
  • Cópia automática (CC): Sempre será enviada uma cópia para o outro usuário envolvido (preenchedor ou aprovador)
  • Fallback automático: Se o usuário não tiver configuração própria, será usado o e-mail padrão do sistema
  • Logs detalhados: O sistema registra qual conta foi usada para cada envio
