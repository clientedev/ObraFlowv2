Como funciona o sistema de aprovação
Aprovador Global:
  • Único em todo o sistema
  • Aprova todos os relatórios sem aprovador temporário
  • Pode transferir o título para outro usuário
  • Pode definir aprovadores temporários
Aprovadores Temporários:
  • Vinculados a projetos específicos
  • Substituem o Aprovador Global para aquele projeto
  • Podem ser removidos pelo Aprovador Global
  • Um por projeto
Prioridade de Aprovação:
  1. Aprovador Temporário do projeto (se configurado)
  2. Aprovador Global (se nenhum temporário)
  3. A aprovação é automática no formulário
